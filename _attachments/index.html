Search

Default search on sample name:

  • "ofhc copper"
  • copper AND ofhc
  • copper OR lead
  • coppe?
  • cop*

Searches are case-insensitive and OR by default.

You can specify a different search field:

  • experiment:majorana
  • technique:(gamma OR naa)
  •    AND name:plastic

These are the supported fields:

  • grouping/experiment
  • name
  • description
  • id
  • source
  • owner
  • institution
  • technique
  • requestor
  • practititoner
  • isotope

To view all assays enter all.

Display

After searching you can click on entries for more detailed information.

  • to see even more detail,
  • to download the assay or report an error,
  • to hide the entry from the display.

You can drag entries into a different order to help you compare them.
